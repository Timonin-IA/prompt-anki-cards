# prompt-anki-cards
LLM prompt for creating cards in Anki

1. Brief description
This tool is used for automated creation of cards in Anki
using LLMs such as ChatGPT, DeepSeek and others. Basically, the author tested
prompt's work in DeepSeek.

2. Use
The prompt is copied and pasted as the first LLM message, with
the variable values changed.
For this version, the list of variables is given below:
- {theme} - card theme
- {number} - the number of cards
- {result} is the format for providing the result, at the moment it is suggested
to leave it unchanged.
